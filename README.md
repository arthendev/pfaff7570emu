# Pfaff 7570 Interface Emulator
Communication Interface emulator of Pfaff Creative 7570, 7550, 1475CD

I created this tool to support:
- reverse-engineering of the communication protocol
- development of a custom stitch design & stitch transfer software
  - enabling work on the project without having the sewing machine connected to PC and avoiding sending gibberish to it
